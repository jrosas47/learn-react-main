# Proyecto de ejemplo: `react-welcome-home`

Este proyecto es un ejemplo sencillo en **React + Vite** para mostrar una **Home Page de bienvenida** y servir como base para explicar conceptos de React.

La página principal muestra:

- Un mensaje de bienvenida
- Una lista de temas que puedes practicar
- Instrucciones rápidas para levantar el proyecto

---

## Requisitos previos

- Tener instalado **Node.js** (que incluye `npm`).
  - Puedes descargarlo desde la página oficial: `https://nodejs.org`
  - Después de instalar, abre una terminal y verifica con:

    ```bash
    node -v
    npm -v
    ```

---

## Pasos para levantar el proyecto

1. Abre una terminal en la carpeta raíz del repositorio:

   ```bash
   cd c:\repositorio\react\learn-react-main
   ```

2. Entra a la carpeta del nuevo proyecto:

   ```bash
   cd react-welcome-home
   ```

3. Instala las dependencias del proyecto:

   ```bash
   npm install
   ```

4. Levanta el servidor de desarrollo:

   ```bash
   npm run dev
   ```

5. Abre tu navegador en:

   ```text
   http://localhost:5173
   ```

Ahí verás la Home Page de bienvenida hecha con React.

---

## ¿Qué puedes hacer con este proyecto?

- Crear nuevos componentes dentro de `src/`
- Modificar `App.jsx` para explicar:
  - JSX
  - Props
  - Estado (state)
  - Eventos y manejo de formularios
- Probar hooks básicos como `useState` y `useEffect`

Este proyecto está pensado como base para **aprender y explicar React** de forma sencilla.
